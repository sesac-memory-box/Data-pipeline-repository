공공데이터 API를 호출하여 데이터를 수집하고, 서비스에 필요한 형태로 전처리한 뒤 백엔드 서버로 전달하는 데이터 파이프라인 레포지토리입니다.

## 역할

- 공공데이터 API 호출
- 원본 데이터 저장
- 데이터 구조 확인
- 데이터 전처리
- 백엔드 서버가 사용할 수 있는 JSON payload 생성
- 백엔드 서버로 데이터 전송

## 기술 선택

현재 데이터 규모가 크지 않고 실시간 스트리밍 처리가 필요하지 않으므로 Kafka, Spark 등의 분산 처리 도구는 사용하지 않습니다.

초기 구현은 Python standard library 기반으로 구성합니다.

## 기본 구조

```text
src/data_pipeline/
├── clients/      # 공공데이터 API 호출
├── preprocess/   # 데이터 정제
├── export/       # 백엔드 전송
├── utils/        # 파일 입출력 등 공통 유틸
├── config.py     # 환경변수 설정
└── main.py       # 실행 진입점
